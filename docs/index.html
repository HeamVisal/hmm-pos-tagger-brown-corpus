How It Works

Model Details

Training Data

The app uses an exported model trained from the bundled Brown corpus files with an 80/20 split and seed 42.

HMM Parameters

Initial, transition, and emission probabilities are estimated from frequency counts. Emissions use Laplace smoothing with alpha 1.0.

Decoder

Viterbi decoding runs in log-space directly in JavaScript, so GitHub Pages can host the app without Python.

Tagset

The model predicts 12 universal tags: ., ADJ, ADP, ADV, CONJ, DET, NOUN, NUM, PRON, PRT, VERB, X.
