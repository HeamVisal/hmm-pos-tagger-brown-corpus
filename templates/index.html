How It Works

Model Details

Training Data

The app loads the bundled Brown corpus files and maps original Brown tags to the universal tagset. It uses the same 80/20 split with seed {{ stats.seed }} as the notebook.

HMM Parameters

Initial, transition, and emission probabilities are estimated from frequency counts. Emissions use Laplace smoothing with alpha {{ stats.alpha }} and an explicit OOV path.

Decoder

Viterbi decoding runs in log-space to avoid underflow. A small epsilon is added before logs, matching the project notebook.
